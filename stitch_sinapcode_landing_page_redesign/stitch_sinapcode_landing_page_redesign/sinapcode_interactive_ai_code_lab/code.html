Implement ReLU

In this exercise, we will implement the Rectified Linear Unit (ReLU) activation function. ReLU is defined mathematically as:

f(x) = max(0, x)

Task Overview

  • Import the necessary NumPy library.
  • Define a function named relu that takes an input array x.
  • Return the element-wise maximum of 0 and x.
