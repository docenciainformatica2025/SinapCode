# Visión General del Sitio: SinapCODE
**ID del Proyecto:** projects/14315644748498424018

## Visión del Sitio
SinapCODE es una plataforma premium para "forjar la próxima generación de Tech Builders". Combina educación, mentoría con IA y seguimiento de evolución gamificado. El sitio está diseñado para sentirse como una herramienta de alta tecnología para profesionales, no solo un LMS estándar.

## Estructura Central
*   **Página de Aterrizaje (`/`):** Introducción de alto impacto, propuesta de valor, "Barra de Confianza", "Hoja de Ruta del Proceso".
*   **Autenticación (`/auth`):** Flujos de Inicio de Sesión y Registro.
*   **Dashboard (`/dashboard`):** Centro para usuarios autenticados. Muestra estadísticas, anillos de progreso y recomendaciones.
*   **Biblioteca (`/library`):** Repositorio de recursos.
*   **Mentor (`/mentor`):** Interfaz de mentoría impulsada por IA.
*   **Evolución (`/evolution`):** Seguimiento avanzado de progreso y estadísticas de gamificación.
*   **Perfil (`/profile`):** Configuración de usuario y gestión de cuenta.

## Contexto Técnico
*   **Framework:** Next.js 14+ (App Router).
*   **Estilos:** Tailwind CSS con una configuración personalizada (ver `DESIGN.md`).
*   **Iconos:** Lucide React.
*   **Fuentes:** Inter & JetBrains Mono.
*   **Estado/Datos:** Supabase (Auth & DB).
